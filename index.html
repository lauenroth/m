What is m

m is a Yeoman generator for Meteor. It helps you scaffolding your app. Why just m and not meteor or something similar? Because it's short.

How to use it?

If you want to create a new project, simply type the following command and m will set up the default folder structure for your new app:

$ yo m

Adding a new view (template + css and javascript) is as easy as that:

$ yo m:view
